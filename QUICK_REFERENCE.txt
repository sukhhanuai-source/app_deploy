## 💾 Database Tables

```
CustomUser
  ├─ User account
  ├─ user_type (worker/admin)
  ├─ phone_number
  ├─ country (FK)
  └─ dashboard_url (FK)

Country
  └─ name

Data
  ├─ name
  ├─ data_type
  └─ country (FK)

DashboardURL
  ├─ name
  └─ url
```

## 🔐 Login Process

```
Visit /login/ 
    ↓
Enter credentials
    ↓
Authenticate
    ↓
Create session
    ↓
Redirect to dashboard
    ↓
Check user_type
    ├→ worker → Show "Hello Family"
    └→ admin → Show "Hi Admin"
```

---

## 📝 Signup Process

```
Visit /signup/
    ↓
Fill form
    ├─ username
    ├─ email
    ├─ password
    ├─ user_type (required!)
    └─ optional: phone, country
    ↓
Create User + CustomUser
    ↓
Auto-login
    ↓
Go to dashboard
```

---

## 🔄 Password Reset

```
Step 1: Forgot Password
/forgot-password/ → Enter email

Step 2: Reset Password  
/reset-password/<token>/ → New password

Step 3: Login
/login/ → Use new password
```

## 📞 Quick Fixes

```

Database error?
→ python manage.py migrate

Can't login?
→ Use testworker/worker123
→ Or testadmin/admin123
→ Or admin/admin

Static files?
→ mkdir static
→ python manage.py collectstatic
```
